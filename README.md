# Flute


##Flute是什么？
* 西洋长笛，[朱痕染迹](http://baike.baidu.com/item/%E6%9C%B1%E7%97%95%E6%9F%93%E8%BF%B9)手持乐器

* App所调API全部源自[新浪微博官方公开API](http://open.weibo.com/wiki/%E5%BE%AE%E5%8D%9AAPI)
